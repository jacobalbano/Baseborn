All included source is 100% original

Controls:
Left and right arrow keys to move
Spacebar to jump
A key shoots a fireball
Pressing the S key starts a beacon that accelerates as it moves. Releasing the key sends a lightning bolt above the beacon
D key sends a frost attack that disables enemies for 3 seconds

When enemies are damaged below half they flee until they get bored, at which point their health regenerates

When all enemies are killed the platform above the exit door moves aside
